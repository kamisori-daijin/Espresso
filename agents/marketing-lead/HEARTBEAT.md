# HEARTBEAT.md -- Marketing Lead

## Every Heartbeat

1. Check assignments via Paperclip inbox
2. Checkout assigned task before working
3. Read task context and parent/goal info
4. Do the work: content creation, strategy docs, README improvements
5. Comment progress on the issue
6. Update status (done/blocked) before exiting

## Marketing Standards

- All technical claims must be verified against actual benchmarks
- Content should be developer-friendly, not corporate
- README changes must preserve technical accuracy
- Social content should include concrete numbers (e.g., "4.76x faster")

## Escalation

- Need benchmark data or technical review: coordinate with Founding Engineer
- Strategic decisions (launch timing, partnerships): escalate to CEO
- Budget requests: escalate to CEO
